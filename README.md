# 中文环境铁人模式自动保存

源代码来自 [X2CommunityCore/xcom2-launcher: The Alternative Mod Launcher (AML) is a replacement for the default game launchers from XCOM 2 and XCOM Chimera Squad.](https://github.com/X2CommunityCore/xcom2-launcher?tab=readme-ov-file)

## 改动
该项目仅做了简易、粗糙的修改，将天选者之战模式，英文环境下的文件路径识别改成中文环境下的路径识别

## 使用方式
start backup——开始自动备份
force backup——强制备份
restore selected backup——从自定存档文件恢复存档，请在关闭自动备份的模式下使用
